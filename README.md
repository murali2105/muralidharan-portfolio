# Muralidharan D Portfolio

Personal portfolio for Muralidharan D, a mechatronics engineering student focused on UAV systems, robotics, embedded electronics, AI vision, fabrication, and VR-based industrial training.

## Highlights

- FORGE VR industrial maintenance training system for Meta Quest 3
- Disaster Management Drone with Edge AI target recognition and payload drop concept
- Delivery Drone and Agriculture Drone project showcases
- Resume-backed achievements, leadership, skills, and contact details

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js star background

## Run Locally

```powershell
npm.cmd install
npm.cmd run dev -- --port 3000
```

Open `http://localhost:3000`.

## Production Build

```powershell
npm.cmd run lint
npm.cmd run build
npm.cmd run start -- --port 3000
```

## Deployment

This app is ready for Vercel or Netlify. Use the default build command:

```powershell
npm.cmd run build
```

For Vercel, import the GitHub repository and deploy as a Next.js project.
